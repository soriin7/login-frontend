Pra rodar a aplicação é necessário rodar um dos comandos abaixo:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

o app consiste em duas telas de login com apenas HTML e CSS.

a telas podem ser acessadas em 

- http://localhost:3000
- http://localhost:3000/idea

Tecnologias utilizadas:

  - React.js
  - Next.js
  - Typescript
  - Tailwind CSS
